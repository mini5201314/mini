# CYBER SNAKE · 赛博贪吃蛇

> 赛博朋克风网页小游戏，双击 HTML 即可运行，无需任何环境。

---

## 游戏说明

玩家操控一条在 20×20 网格中穿行的霓虹蛇，吞食发光的数据球获得分数。

### 操作方式

| 按键 | 功能 |
|------|------|
| `↑` `←` `↓` `→` / `W` `A` `S` `D` | 控制移动方向 |
| `P` | 暂停 / 继续 |
| `空格` | 开始 / 重新开始 |

### 规则

- **穿墙**：碰到边界自动从另一侧出现
- **自撞**：碰到自身身体游戏结束
- **普通食物**：每个 +10×等级 分
- **特殊食物**（粉色菱形）：每 50 分概率出现，加 +30×等级 分，带计时消失
- **升级**：每 100 分升一级，速度加快（10 级封顶）

### 计分

- 当前分数、本局最高等级实时显示在 HUD 面板
- 最高分自动保存至浏览器本地存储，重开不丢失

### 技术实现

- 纯前端实现，无任何依赖，双击直接运行
- Canvas 2D 渲染，逐帧绘制粒子系统与霓虹光效
- 触摸端支持滑动控制
